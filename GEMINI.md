# AI Devs 4 Project

## Project Overview

This project, `ai_devs4`, is a repository for completing tasks related to the "AI Devs 4" course or project. 

### Main Technologies

- **Node.js**: The tasks will be solved using Node.js 
- **JavaScript**: The source code is written in JavaScript.

## Development Conventions

- This is a **minimalistic** JavaScript project.
- Tasks are organized by episode (e.g., `S01E01`).
